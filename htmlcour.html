1HTML

aDEFINITION

Le HTML ( HYPERTEXT MARKUP LANGUAGE ) est le format de donnée conçus pour présnter des pages web qui pouvant etre lues par des navigateurs

(EXEMPLES:netscape,internet explorer,opera...)

C'est un langage de présentation de données et non pas un langage de programmation

bSTRUCTURE D'UNE PAGE HTML

cAVANTAGE DU HTML

-Facilite la réalisation de lien d'un serveur vers un autre

-Le texte HTML peut etre écrit avec un éditeur standard

-On peut réaliser assez facilement un serveur

dINCONVENIENTS DU HTML

-Le langage HTML n'est pas assez complet

-Melange entre la présentation et le contenu

-Les possibilites de mise en forme sont limites et parfois complexe a mettre en oeuvre (FRAMES)

-Le stockage d'une page réprésente en general plusieur fichiers

fSTRUCTURE D'UNE PAGE WEB 2CSS aDEFINITION

Les feuilles de style en cascarde CSS ( Cascarding Style Sheets ) est un langage informatique qui sert a decrire la présentation des documents

HTML,XHTML et XML.Les standards définissant CSS sont publiés par le W3C (WORL WIDE CONSORTIUM) bAVANTAGE DU CSS

-La structure du document et la presentation peuvent etre gerées dans des fichiers séparés

-La concption d'un document se fait dans un premier temp sans se soucier de la presentation ce qui permet

d'etre plus efficace.

-Un meme document peut donner le choix entre plusieur feuilles de style,par exemple une pour l'impression et une pour la lecture a l'ecran

-Le code HTML est considerablement reduiten taille et en complexite ,puisqu'il ne contient plus de balise ni d'attribut de presentation

élégants et bien organises.Les CSS comme toutes les technologies de developpement web,évoluent au meme rythme que les navigateurs,ceci nous invite a s'aligner avec toutes ces évolution toutes ces évolutions tout en profitons des bonnes pratiques de desveloppement web
